Read ME:
==============

1. Open the Assignment01.ipynb in jupyter notebook 
2. Run all the cells
3. Before running the testing cells(Near end of ipynb file) you need to specify the read_folder and write_folder.
4. Also you need to have model_weights in order to run the model.
5. Some parts of the code has been commented out and they are only needed for training.
6. You need the array of images and labels as inputs to be loaded as well.
